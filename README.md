## Installation

$ yarn install

## development mode

$ yarn start:dev

## local mode

$ yarn start:local

## kfund24 admin 폴더 구조

- src 폴더
- src/assets: 이미지, 폰트 등
- src/constants: 상수
- src/pages: 페이지 단위 컴포넌트
- src/components: 기타 컴포넌트
- src/contexts: context api 모음
- src/modules: 네이티브 모듈
- src/store: 스토어 세팅
- types: 타입 정의
